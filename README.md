# Goobo-Market – Bakhaarkaaga Casriga Ah

Goobo-Market waa nidaam e-commerce oo loogu talagalay in lagu iibiyo alaabta noocyadeeda kala duwan. Waxaa lagu dhisay Node.js, Express, iyo MongoDB.

## Tilmaamaha Muhiimka Ah
- **Maamulka Alaabta**: Soo bandhigidda alaabta noocyo kala duwan ah.
- **Dambiisha (Cart)**: U oggolaanshaha macaamiisha inay alaabta dambiisha ku ritaan.
- **Dalbashada (Checkout)**: Nidaam fudud oo alaabta lagu dalban karo.
- **Cinwaanka Maamulka**: Dashboard loogu talagalay in lagu daro ama laga tirtiro alaabta.

## Sida Loo Bilaabo (Setup)

1. **Rakibidda Dependecies-ka**:
   ```bash
   npm install
   ```

2. **Habaynta .env**:
   Sameyso file la yiraahdo `.env` oo ku dar `MONGO_URI` iyo `PORT`.

3. **Bilaabista Server-ka**:
   ```bash
   npm run dev
   ```

## Folder Structure (Qaabka Files-ka)
- `backend/`: Dhammaan code-ka backend-ka iyo API-ga.
- `frontend/`: Bogagga HTML, CSS, iyo JavaScript-ka.

## Tech Stack
- **Backend**: Node.js & Express
- **Database**: MongoDB & Mongoose
- **Frontend**: HTML5, CSS3, iyo Vanilla JavaScript
